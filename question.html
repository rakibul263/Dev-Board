Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select an element in the DOM. Some of the most common ways are:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the HTML content inside an element, including HTML tags

innerText: Gets or sets the visible text content, respecting CSS styles.

textContent: Gets or sets the text content, including hidden text, but ignores HTML tags and styles.

Question-3: What is event delegation in the DOM?

Event delegation is when you attach a single event listener to a parent element, and it handles events for child elements. This reduces the number of event listeners and improves performance.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts at the target element and then "bubbles up" to its parent elements, triggering any event listeners on those parents. It allows you to handle events at a higher level in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?
